# 一期页面清单与落地状态

## 仓库端

- `GET /login` 登录页（已实现）
- `GET /` 工作台（已实现）
- `GET /finance` 财务管理（已实现：收入/支出/采购录入与提交、利润快照）
- `GET /finance/analysis` 财务分析（已实现：AI分析与合伙人分账）
- `GET /kpi` KPI 管理（已实现：填报、规则草稿/发布、结果列表）
- `GET /kpi/analysis` KPI 分析（已实现：指标得分下钻）
- `GET /demands` 需求中心（已实现：创建、列表、详情、确认完成）

## 管理端

- `GET /admin` 管理员总览（已实现：需求审核、KPI审批、利润重算）

## 一期页面对应核心能力

- 财务主线：录入 -> 提交 -> 管理员审核 -> 快照/分析
- KPI主线：填报 -> 提交 -> 管理员审核 -> 结果/分析
- 需求主线：发起 -> 审核 -> 跟进 -> 完成/关闭

## 备注

- 本版本保持双角色：`warehouse`、`admin`
- 按钮级权限通过后端角色守卫 + 前端业务状态控制共同约束
